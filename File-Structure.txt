This is the file telling about the structure of data stored in txt files

1- Users.txt file contains 2 items i.e username , password per record (Line) stored in the structure as :
username password

2- Items.txt file contains data of the items available and per record (line) contains following data items i.e category ID , category , itemname , available stock , price as following structure:
catrgoryID category itemname availableStock price

3- Orders.txt contains data of the orders done and contains following data items i.e username , delivery address , contact number , payment method and items brought in the structure
username totalPrice deliveryAddress contactNumber paymentMethod item1,item2,item3......

4- CurrentUser.txt file only contains the username of currently logged in user and is created at time of login and is deleted at time of logout
